Ce formulaire vous aidera à visualiser ou convertir un fichier .xml généré par le logiciel apafis en un fichier word .doc.

Vous avez deux possiblités :

  • Convertir le fichier xml en fichier doc et le télécharger

  • Afficher le fichier xml en page web lisible dans le navigateur.

Les documents envoyés sont supprimés au bout d'une minute automatiquement.
